# AI Restaurant Assistant

A premium, voice-driven AI system for high-end restaurants.

### Features
- Natural Language Interpretation
- Autonomous Cart Management
- Multi-Category Premium Menu
- Advanced Customization System
- Real-time AI Status Messaging

### Tech Stack
- Frontend: React 18, Vite, Lucide Icons
- Backend: Python Flask, Gemini AI Integration
- Styling: Vanilla CSS (Custom Design System)

### Setup
1. Run backend/server.py
2. Run npm install && npm run dev in frontend/
