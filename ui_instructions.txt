Design Theme:
- Light Minimal Café

Core Feel:
- Warm
- Premium
- Friendly
- Modern

Experience Goals:
- User should feel relaxed and welcomed
- UI should feel clean but not cold
- Interaction should feel effortless and smooth

Visual Style Rules:
- Use light backgrounds (no dark UI)
- Avoid harsh contrasts (no pure black on pure white)
- Prefer soft tones over bright or neon colors
- Keep interface breathable (high spacing, low clutter)

Design Personality:
- Not playful
- Not corporate
- Not tech-heavy
- Balanced between café warmth and modern simplicity

Clarity Rules:
- Every element must have a purpose
- No decorative elements without function
- No visual noise

Hierarchy Rules:
- Important actions must stand out clearly
- Secondary elements must be subtle
- Text readability is highest priority

AI Restrictions:
- Do NOT introduce new styles outside this theme
- Do NOT use vibrant or flashy colors
- Do NOT create dense or crowded layouts

Color Philosophy:
- Warm light background
- Soft contrast
- Premium café tone
- No harsh black-white contrast

Voice States:

1. Idle
2. Listening
3. Processing
4. Responding

Each state must have clear but subtle visual feedback

--------------------------------------------------

1. Idle State:

- Mic icon visible (not dominant)
- No animation
- Soft presence

Text (optional):
- "Tap or speak to order"

--------------------------------------------------

2. Listening State:

Trigger:
- User taps mic OR starts speaking

UI Changes:
- Mic becomes active
- Soft pulse animation

Animation Rules:
- Slow breathing effect
- Scale: 100% → 110%
- Duration: ~1.5s loop
- No harsh glow
- No neon effect

Transcript:

- MUST be visible
- Real-time text display
- Positioned near top or center

Transcript Style:
- Font: Outfit
- Size: 16px
- Color: #1E1E1E
- Background: none or very subtle

Behavior:
- Updates word-by-word
- Handles pauses gracefully

--------------------------------------------------

3. Processing State:

Trigger:
- User stops speaking

UI Changes:
- Mic animation stops
- Show subtle loader

Loader Style:
- Minimal (dots or fade)
- No spinner overload

Text:
- "Understanding..." (optional)

Duration:
- Keep short and smooth

--------------------------------------------------

4. Responding State:

Trigger:
- AI gives response

UI Changes:
- Show AI message text

Style:
- Clean text block
- No chat bubbles (keep minimal)

Text Style:
- Font: Outfit
- Size: 16px
- Color: #444444

Optional:
- Soft fade-in animation

--------------------------------------------------

Voice Feedback Rules:

- Always show transcript (build trust)
- Keep animations subtle
- Avoid loud visual effects

--------------------------------------------------

Error Handling:

If voice unclear:

UI:
- Show transcript with issue
- Example: "Did you mean paneer pizza?"

Behavior:
- Suggest correction
- Allow retry

--------------------------------------------------

Noise Handling:

- Show what AI heard
- Allow quick correction via tap
- Never assume silently

--------------------------------------------------

Accessibility:

- Always allow manual fallback (buttons)
- Voice is primary, not exclusive

--------------------------------------------------

AI Restrictions:

- Do NOT add heavy animations
- Do NOT use glowing neon effects
- Do NOT create chat-style UI
- Do NOT hide transcript
- Do NOT overload screen with voice visuals
